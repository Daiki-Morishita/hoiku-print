## 1. 戦略方針

### 1.1 ポジショニング

**「ぬりえで集客 → 知育プリントで滞在・回遊させるサイト」**

- 集客の主入口は引き続き「ぬりえ」（月間検索165K・最強のヘッドKW）
- 滞在・PV・リピートは「知育プリント」（ひらがな・迷路・点つなぎ等）で稼ぐ
- ドメイン・サイト名は変更しない（nurie-print.com / ぬりえプリント）
- タグライン・ナビ・titleで「知育も扱う」ことを明示

### 1.2 競合との差別化

| 競合 | 強み | 弱み（攻めどころ） |
|---|---|---|
| ちびむすドリル | 素材量・SEO圧倒 | ぬりえカテゴリは薄い・UIが古い |
| ぷりんときっず | プリント品質・キャラ可愛い | 広告多い・モバイル弱い |
| ぬりえやさん／ぬりえラボ | ぬりえ特化 | 知育に展開していない |

→ **「ぬりえの質 × 知育の網羅性 × モダンUI/UX × モバイル最適化」** で勝負する。

## 3. URL構造

### 3.1 現状

```
/                          トップ
/materials                 全教材一覧
/materials/[id]            教材詳細
/category/[type]/[value]   カテゴリ別一覧（theme/category/age）
```

### 3.2 知育拡張後の推奨構造

```
/                          トップ（カテゴリ別セクションで構成）
/coloring                  ぬりえ一覧（旧 /materials の主役）
/coloring/[theme]          ぬりえ × テーマ別（animals 等）
/coloring/[id]             ぬりえ詳細
/hiragana                  ひらがな一覧
/hiragana/[set]            50音 / 単語 / 濁音 等
/hiragana/[id]             ひらがな詳細
/maze                      迷路一覧
/maze/[difficulty]         難易度別
/maze/[id]                 迷路詳細
/numbers                   数字一覧
/dotconnect                点つなぎ一覧
/drawing                   運筆一覧
/column                    コラム
/column/[slug]             コラム詳細
```

**設計原則**：
- カテゴリ名をパス第一階層に置く（SEO・人間可読性）
- 旧 `/materials/[id]` は当面リダイレクトでサポート
- 旧 `/category/theme/animals` も `/coloring/animals` へ301

## 4. ユーザー動線（Conversion Path）

### 4.1 メインフロー

```
[流入] Google「ぬりえ 無料」
   ↓
[着地] /coloring または /coloring/[theme]
   ↓
[詳細] /coloring/[id]   ← 印刷ボタンの近くに関連知育プリント
   ↓
[回遊] 「このどうぶつテーマで ひらがな練習」「迷路で続けて遊ぶ」CTA
   ↓
[2枚目印刷] /hiragana/animal-words または /maze/easy
   ↓
[ブックマーク・お気に入り] 会員化（v3で実装予定）
```

### 4.2 詳細ページに必須の回遊セクション

ぬりえ詳細ページ下部:
- **同テーマの知育プリント**: 動物ぬりえ → 動物ひらがな・動物点つなぎ
- **次の難易度**: simple → easy → normal → rich
- **同シリーズの別バリアント**: variant-1 → variant-2

知育プリント詳細ページ下部:
- **同テーマのぬりえ**: 学びの後の塗り絵で「ごほうび」
- **次のステップ**: ひらがな50音 → ひらがな単語 → カタカナ

## 5. データモデル拡張

### 5.1 既存 Material 型の拡張

```typescript
type Material = {
  id: string
  title: string
  description: string
  category: Category    // coloring / hiragana / numbers / maze / ...
  theme?: Theme         // animals / vehicles / ... (横断テーマ)
  difficulty: 1|2|3|4   // simple / easy / normal / rich
  ageMin: number
  ageMax: number
  // ...既存フィールド

  // 知育拡張で追加
  contentType: 'image' | 'svg' | 'pdf' | 'interactive'  // 表示形式
  generationMethod: 'chatgpt' | 'algorithm' | 'font' | 'manual'  // 量産方法
  printSize: 'A4-portrait' | 'A4-landscape'
  relatedIds?: string[]    // 同テーマの他カテゴリ素材ID
  prerequisiteIds?: string[]  // 前提となる素材（数字10→数字20）
}
```

### 5.2 横断テーマの考え方

`theme` は category をまたぐタグ。例：
- `animals` テーマで coloring・hiragana・dotconnect を横断
- ユーザーが「動物が好き」→ 動物のぬりえ・動物ひらがな・動物点つなぎを連続提供

## 7. SEO戦略

### 7.1 カテゴリ別 title / description 雛形

| カテゴリ | titleパターン | description要旨 |
|---|---|---|
| /coloring | 無料ぬりえ｜動物・乗り物・恐竜の塗り絵プリント\|ぬりえプリント | 印刷無料・登録不要 |
| /hiragana | ひらがな練習プリント無料\|なぞり書き・五十音\|ぬりえプリント | 2歳〜6歳対応 |
| /maze | 迷路プリント無料\|簡単〜難しい\|2歳〜6歳\|ぬりえプリント | 自動生成で素材多数 |
| /coloring/[id] | {タイトル}\|無料ぬりえダウンロード\|ぬりえプリント | テーマ・難易度・年齢 |

### 7.2 構造化データ追加

- 各カテゴリページに `BreadcrumbList` + `ItemList`
- 個別ページに `CreativeWork`
- コラムに `Article` + `FAQPage`
- サイト全体に `WebSite` + `Organization`（既存）

### 7.3 内部リンク戦略

- カテゴリページ間相互リンク（フッター + 詳細ページ下部）
- テーマ横断リンク（動物ぬりえ ↔ 動物ひらがな）
- コラムから関連素材への自然導線
