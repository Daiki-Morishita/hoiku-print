# 用語定義（共通言語）

このプロジェクト内で使う言葉の定義。スクリプト・コード・会話すべてでこの定義に従う。

---

## 素材の構造

```
カテゴリ
  └─ テーマ
       └─ アイテム
            └─ バリアント（-1 / -2 / -3）
                 └─ ユニット（simple / easy / normal / rich）
```

| 用語 | 説明 | 例 |
|---|---|---|
| **カテゴリ** | 教材の種類 | ぬりえ、ひらがな、迷路 |
| **テーマ** | カテゴリ内の大分類 | 恐竜、公園、動物 |
| **アイテム** | テーマ内の個別題材 | ブランコ、ティラノサウルス |
| **バリアント** | 同アイテムの異なる場面構成（-1/-2/-3） | swing-1、swing-2 |
| **ユニット** | 難易度別の1枚（1バリアントに4枚） | swing-simple-1 |

## 難易度（`difficulty` フィールド）

| 値 | ID | 日本語名 | 対象年齢 | 構図 |
|---|---|---|---|---|
| 1 | `simple` | やさしい | 2〜3歳 | 対象1つのみ・背景なし・余白たっぷり |
| 2 | `easy` | ふつう | 3歳 | シンプルな背景つき |
| 3 | `normal` | むずかしい | 4〜5歳 | 複数・場所がわかる背景 |
| 4 | `rich` | とてもむずかしい | 4〜6歳 | 家族・にぎやかな背景 |

---

## ファイルID命名規則

```
{アイテムID}-{難易度ID}-{バリアント番号}
例: swing-simple-1 / tyrannosaurus-rich-2
```
